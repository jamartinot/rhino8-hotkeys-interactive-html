Rhino 8: Full Beginner's Course Reference

Chronological Visual Cheat Sheet & Technical Guide

Part 1: UI, Navigation & Moving Geometry

ActionInputsWhat Happens (Output)Why Use It?
Orbit 🖱️Right-Drag Camera rotates continuously around the viewing center. To inspect the 3D model from different angles.
Pan Shift + 🖱️Right-Drag Camera slides left/right or up/down without rotating. To shift your field of view across the workspace.
Zoom Ctrl + 🖱️Right-Drag
or Scroll Wheel		 Camera scales in/out smoothly. To get closer to fine details or see the whole scene.
Zoom Selected Zs + Enter Viewport centers directly on the currently selected object. To find lost geometry or rotate specifically around one part.
Toggle Viewport 🖱️Double-Click Name Expands one viewport to full screen, or returns to 4-view. To switch between working in 2D (Top/Front) and 3D (Persp).
Gumball Toggle button at bottom Widget appears on object: Arrows (Move), Arcs (Rotate), Squares (Scale). For fast, intuitive transformations without typing commands.
Move Move + Enter Relocates geometry from a precise Point A to Point B. When Gumball isn't accurate enough; ensures perfect snapping.
Orient 3-Point Orient3Pt + Enter Aligns an object by matching 3 source points to 3 target points. To accurately place complex objects on angled surfaces.

Part 2: Basic Geometry & Curves

CommandInputsWhat Happens (Output)Why Use It?
Points On F10 Control points/vertices become visible and selectable. To manually deform the shape of a curve or single surface.
Points Off F11 Control points are hidden. To lock in edits and return to normal object selection.
Sub-Select Ctrl + Shift + 🖱️Click Selects a single face, edge, or point of a joined polysurface. To modify a specific part of a solid without exploding it first.
Join / Explode Join / Explode Connects touching edges into one unit / breaks units apart. To prep curves for extruding, or to break a box into flat planes.
Extrude Curve ExtrudeCrv + Enter Pulls a 2D line into a 3D surface (open) or solid (closed). To give 2D sketches depth (e.g., floorplan to walls).
Cap Cap + Enter Fills flat, planar open holes on polysurfaces. To turn an open shape into a "watertight" solid for 3D printing.

Part 3: Geometry Types (NURBS, Mesh, SubD)

ConceptInputs / CommandsWhat Happens (Output)Why Use It?
NURBS Rebuild Math-driven geometry. Outputs infinitely smooth curves/surfaces. Default for CAD. Best for exact manufacturing and hard edges.
Meshes Mesh Tools panel Geometry made of flat polygon faces (vertices/edges). Used for 3D scanning or exporting to game engines/zBrush.
SubD Tab (Toggles smooth/box) Organic, clay-like geometry. Extrude faces with Ctrl+Drag. Best for highly organic, continuous forms (furniture, creatures).

Part 4: Layers & Materials

CommandInputsWhat Happens (Output)Why Use It?
Revolve Revolve + Enter Spins a profile curve around an axis to make a 3D surface. To make cylindrical/symmetrical objects like lightbulbs or vases.
Mirror Mirror + Enter Creates an exact replica across a defined line. To build symmetrical objects instantly without drawing twice.
Change Layer 🖱️Right-Click Layer ➔ Change Object Layer Moves selected geometry to the specified organizational layer. To organize complex files, allowing you to hide/lock parts easily.

Parts 5-8: 2D to 3D, Modeling Aids & CPlanes

Command / AidInputsWhat Happens (Output)Why Use It?
Sweep 1 / 2 Sweep1 / Sweep2 Extrudes a shape along 1 or 2 guide rails. To make complex piping, molding, or custom structural frames.
Curve Boolean CurveBoolean Combines overlapping curves by clicking inside the desired regions. Faster than manually trimming multiple overlapping 2D shapes.
Ortho Mode Hold Shift while drawing Constrains mouse movement to strict 90° or 45° angles. To ensure lines and movements are perfectly straight.
Nudge Alt + Arrow Keys Steps object by tiny, exact increments. (Shift = large step). To micro-adjust placement without losing grid alignment.
Project Project / ProjectToCPlane Throws a curve onto a 3D surface, or flattens it to the floor. To map 2D logos onto curved objects, or fix non-flat curves.
Select Dupes SelDup + Enter Selects identically overlapping geometry. To clean up messy files and prevent 3D printing errors.

Parts 9-10: Advanced Transforms & Solid Edits

CommandInputsWhat Happens (Output)Why Use It?
Rotate 3D Rotate3D + Enter Rotates geometry around a custom drawn axis line. To rotate parts (like hinges) that are not aligned to the grid.
Scale 1D / 2D Scale1D / Scale2D Stretches an object in only 1 or 2 directions based on points. To stretch a pillar exactly to a ceiling without making it wider.
Split / Trim Split / Trim Split cuts into pieces; Trim cuts and deletes the excess. To carve up surfaces using intersecting lines or other planes.
Boolean Union BooleanUnion Fuses intersecting solids into one continuous, watertight mesh. To merge parts for final manufacturing.
Boolean Diff BooleanDifference Uses Solid A to cut a hole out of Solid B. To create sockets, screw holes, or intricate hollows.
Fillet Edge FilletEdge Rounds off the sharp 90-degree corners of a solid volume. To make realistic, handleable objects (removes sharp edges).

Part 11: Arrays, Flow & Cage Deformations

CommandInputsWhat Happens (Output)Why Use It?
Array ArrayLinear / ArrayPolar Creates precise copies in lines, 3D grids, or circular patterns. To generate stairs, gears, or patterned architecture instantly.
Flow Along Srf FlowAlongSrf Morphs a flat 3D object to wrap around a curved target surface. To apply complex textures, scales, or text to organic shapes.
History History (Record = Yes) Links generated geometry to its original parent curve. Updating the base curve automatically updates the 3D model.
Deformations Twist / Bend / Taper Applies global, sweeping bending or twisting forces to a solid. To turn straight geometry into dynamic, sweeping forms.
Cage Edit CageEdit Wraps objects in a control box with editable points. To organically push/pull complex groups of objects seamlessly.
